简答

1.java什么时候用重载，什么时候用重写？
      遵循开闭原则，Spring中的getBean，可以传BeanName和Class
      父子类之间，模板和包装器模式，遵循开闭原则
2.举例一个更倾向于使用抽象接口类，而不是使用接口的设计场景？
      抽象类可以增加实现，接口不能增加实现。抽象类可以有用在public方法中调用private方法，防止private方法被子类实现
3.在java中，为什么不允许从静态方法中访问非静态变量？
      类的静态成员(变量和方法)属于类本身，在类加载的时候就会分配内存，可以通过类名直接去访问；非静态成员（变量和方法）属于类的对象，所以只有在类的对象产生（创建类的实例）时才会分配内存，然后通过类的对象（实例）去访问。在一个类的静态成员中去访问其非静态成员之所以会出错是因为在类的非静态成员不存在的时候类的静态成员就已经存在了，访问一个内存中不存在的东西当然会出错。
4.软件架构中的上层应用指哪些层，客户端属于上层应用么？
      架构有前端层，后端层，中间层，客户端是最顶层就是上层，如果在中间层或后端层，就可能不是

面试

1.受检异常与不受检异常的区别
      受检异常必须要处理，比如SQLException,IOException
      不受检异常不是必须要捕获
2.了解软引用，弱引用，虚引用，强引用
      强引用被回收的情况:1>脱离作用域;2>设置为null,并不再被引用

   public class ReferenceDemo{
       static Object strongRef = new Object();
       public static void main(String args[]){
           Object obj = strongRef;
           strongRef = null;
           System.gc();
           System.out.println("gc之后:"+obj);
           //gc之后java.lang.Object@4b1210ee,此时因为obj还引用了strongRef所以不会被回收
       }
   }

      软引用在发生OOM异常前才会被回收

   public class ReferenceDemo{
       public static void main(String args[]){
          Object softRef = new Object();
          SoftReference softReference = new SoftReference(softRef);
          System.out.println(softReference.get());
          softRef = null;//未被回收
          System.out.println(softReference.get());
          softReference = null;//被回收
          System.out.println(softReference.get());
       }
   }

      弱引用的对象没有被使用会被回收掉,与对象声明周期有关

   public class ReferenceDemo{
       public static void main(String[] args){
           String str ="aa";
           //Object weakObject = new Object();
           //WeakReference weakReference = new WeakReference(weakObject);
           //weakObject = null;//被回收
           //System.gc(); 去掉gc() 不被回收
           //System.out.println("after:"+weakReference.get());//null
           WeakReference weakReference = new WeakReference(str);
           str = null;
           System.gc();
           //gc()不回收常量池
           System.out.println("after:"+weakReference.get());//"aa"
       } 
   }

      虚引用 get()拿不到对象 只有回收前取队列中数据才能取到。

   public class ReferenceDemo{
       public static void main(String[] args[]){
           ReferenceQueue queue = new ReferenceQueue();
           Object phantomObject = new Object();
           PhantomReference phantomReference = new PhantomReference(phantomObject,queue);
           System.out.println(phantomReference.get());//null
           phantomObject =null;
           System.gc();
           System.out.println(phantomReference.get());//null
           Thread.sleep(200);
           System.out.println(queue.poll());//java.lang.ref.PhantomReference@..    
       }
   }
